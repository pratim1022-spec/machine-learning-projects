🎬 Movie Recommendation System

This project builds a Movie Recommendation System that suggests movies to users based on similarity between movies.  
It demonstrates a complete machine learning workflow — from data preprocessing and feature extraction to similarity computation and recommendation generation — using Python and Jupyter Notebook.

■ Project Overview:-

Implements a content-based movie recommendation system

Uses movie metadata such as genres, keywords, cast, and overview

Computes similarity between movies using cosine similarity

Recommends movies similar to a selected movie

Provides a simple and efficient way to discover related movies

■ Technologies Used:-

Python

pandas

numpy

scikit-learn

nltk

matplotlib / seaborn

Jupyter Notebook

■ Files:-

Movie_recommendation.ipynb — main notebook

README.md — project documentation

movies.csv / dataset file — movie metadata dataset

■ How to Run:-

Clone this repository

Open the notebook in Jupyter

Run all cells to build the recommendation model

Enter a movie name to get recommended movies

■ Results:-

The system recommends top similar movies based on cosine similarity

Demonstrates how content-based filtering works in recommendation systems

Provides quick and relevant movie suggestions
